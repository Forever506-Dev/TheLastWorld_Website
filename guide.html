How To Join The Server

1. Install GTA V and FiveM

  1. Install Grand Theft Auto V (Steam, Rockstar, or Epic) and run it once.
  2. Download the FiveM client from the official Cfx.re website and install it.
  3. When FiveM starts, choose your GTA V folder if asked and let it update.

2. Connect To The Server

  1. Launch FiveM, go to the Servers or Direct Connect tab.
  2. Search for YOURSERVERNAME or use direct IP your.ip:30120.
  3. Click Connect and accept any resource downloads.

3. Recommended Settings

  • Disable overloaded visual mods and heavy reshades to avoid crashes.
  • Use a wired connection when possible for better sync during horde events.
  • Join our Discord for support, whitelist, and modpack updates.
